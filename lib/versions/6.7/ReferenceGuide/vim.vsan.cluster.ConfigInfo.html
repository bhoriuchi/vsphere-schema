Data Object Description

The VsanClusterConfigInfo data object contains configuration data for the VSAN service in a cluster. This data object is used both for specifying cluster-wide settings when updating the VSAN service, and as an output datatype when retrieving current cluster-wide VSAN service settings.
See ReconfigureComputeResource_Task
