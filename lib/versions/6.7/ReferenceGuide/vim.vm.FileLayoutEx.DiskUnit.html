Data Object Description

Information about a single unit of a virtual disk, such as the base-disk or a delta-disk.

A disk-unit consists of at least one descriptor file, and zero or more extent files.

Sometimes, a disk-unit is also referred to as a backing.

Properties

Name Type Description
fileKeyxsd:int[]

Array of keys of the files that make up the disk unit. Values here correspond to property key in file.

At least one entry always exists in this array. Property type of the referenced file can be used to distinguish the disk descriptor (type diskDescriptor) from the extents.
