System Counters

Performance counters are modeled using the PerfCounterInfo data object type. Each instance of a PerfCounterInfo data object contains metadata about a specific metric available from the system.

The system group of counters support metrics for overall system availability. These counters are available directly from ESX and from vCenter Server.
