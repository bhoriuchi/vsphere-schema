Data Object Description

This data object type describes a request to invoke a specific method in a VMOMI service. It currenly only supports {link vim.SessionManager#cloneSession} method. The GenericServiceTicket.id returned from AcquireGenericServiceTicket for this request can be use for CloneSession to clone a session
