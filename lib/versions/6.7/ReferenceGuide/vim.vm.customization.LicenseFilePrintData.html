Data Object Description

The LicenseFilePrintData type maps directly to the LicenseFilePrintData key in the sysprep.xml answer file. These values are transferred into the sysprep.xml file that VirtualCenter stores on the target virtual disk. For more detailed information, see Performing Unattended Installations. LicenseFilePrintData provides licensing information for Windows server operating systems.
